The source logo badge (top right) is a custom slot — it receives currentStream as a prop and you render whatever you want. Addon icon, quality badge, provider name — all yours.

How source logos plug in — custom slot
// Player.tsx — drop-in custom slot <MediaPlayer src={streamUrl}> <MediaProvider /> <DefaultVideoLayout slots={{ // Top-right corner — your source logo topControlsGroupEnd: ( <SourceBadge stream={currentStream} /> ), // Below scrubber — extra info row beforePlayButton: ( <EpisodeLabel title={title} /> ), // Bottom right — extra buttons afterSettingsMenu: ( <AirPlayButton /> ), }} /> </MediaPlayer> // SourceBadge.tsx — renders addon logo + quality function SourceBadge({ stream }) { const logo = ADDON_LOGOS[stream.addonId]; return ( <div className="source-badge"> {logo && <img src={logo} alt={stream.addonName} />} <span>{stream.addonName}</span> </div> ); }
Available slot positions
topControlsGroupStart / End
Top bar left and right — title, back button, source logo
✓ used for logo
beforePlayButton / afterPlayButton
Either side of the play button in the center row
afterCaptionButton / afterAudioGainSlider
Insert custom buttons anywhere in the control bar
settingsMenuStartItems
Add rows to the settings menu (e.g. "Switch source")
