Search quality improvements (code changes):

• Debounce suggestions: 250ms debounce on input → calls fetchMeta-style search on the top catalog with the query as genre/search extra
• Recent searches: Stored in localStorage, max 5 items, shown on empty state
• Trending on empty: Pulled from the existing Popular Movies + Popular TV rows already loaded on home (passed via a shared store or re-fetched from the manifest)
• Suggestions dropdown: Shows up to 4 quick results with poster thumb + type + year. Clicking navigates directly to detail page
• Better ranking: Deduplicate by ID, sort by popularity desc, prefer exact title matches at top
• Filter chips: Client-side filter on already-fetched results (no extra API calls)
