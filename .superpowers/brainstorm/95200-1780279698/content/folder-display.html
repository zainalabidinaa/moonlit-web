A — Folder Grid on Home

Folders appear as 4-column poster grids inline on the home screen. Each folder shows its cover posters. Tapping a folder navigates to its browse page showing all content.

B — Collection Banner Row

Folders are grouped by collection and shown as wide landscape tiles in a horizontal scroll row, labeled with the collection name. Tapping a tile goes to that folder's browse page.

C — Home stays clean, folders go to Library/Browse tab

Home screen only shows the 4 featured rows. All other collections/folders live in the Library or Browse section. Home is lean and fast; discovery is a dedicated experience.
